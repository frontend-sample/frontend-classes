Treinamento Básico de Frontend

Nesse treinamento iremos criar um tipo especial de site chamado Landing Page que é comumente utilizada para anunciar um novo produto, serviço, empresa, etc e é uma página web focada em conversões, projetada para capturar leads ou promover uma ação específica (como baixar um e-book, inscrever-se em uma newsletter, etc.). Ela é otimizada para direcionar e engajar visitantes, geralmente com um design simples e um call-to-action claro.

Se você quiser saber mais sobre landing pages esse artigo pode te ajudar.

Esse tipo de site costuma ter apenas uma página com várias sessões e durante seu desenvolvimento vamos aprender a base do HTML e do CSS.

Iremos salvar todo o código do curso no github. Os passos serão explicados nas próximas aulas.

Não se esqueça de sempre enviar seu código ao github.

Em breve falaremos sobre o Git Flow que uma abordagem de como usar e padronizar branches manter um melhor controle e entendimento do seu fluxo de mudanças
